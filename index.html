How the layers work

Library: webshare-transfer.js. Moves arbitrary JSON-serializable payloads peer-to-peer via PeerJS or Raw WebRTC + QR. No UI, no DOM, no opinion on payload contents. Exposes a small event-driven API.

App: this demo page. Owns the session data model, IndexedDB schema, conflict resolution UI, session list. Calls the library to move payloads. When a payload arrives, the demo shows a dialog asking what to do.

For your real app (e.g. AVScout): import webshare-transfer.js, plug in your own data model and UI. The library doesn't care whether you're transferring assets, photos, audits, or anything else.
